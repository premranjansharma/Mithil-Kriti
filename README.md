# Mithila Kriti

Mithila Kriti is a premium e-commerce platform showcasing authentic Mithila (Madhubani) art, handcrafted products, traditional clothing, and unique gifting collections.

## Features

* Responsive Next.js 16 Application
* Modern UI with Tailwind CSS
* Product Categories
* Shopping Cart
* Wishlist
* User Account Section
* Mobile-Friendly Navigation
* SEO Optimized Structure

## Tech Stack

* Next.js 16
* React
* TypeScript
* Tailwind CSS
* Vercel Deployment

## Installation

Clone the repository:

```bash
git clone https://github.com/premranjansharma/Mithil-Kriti.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

http://localhost:3000
